# Sprint1-SERS-ProjetoSustentavel

CollectEV Infraestrutura Inteligente de Recarga para Condomínios Conectados Sobre o Projeto

O CollectEV é uma solução de infraestrutura inteligente voltada para a recarga coletiva de veículos elétricos em condomínios.

O projeto transforma carregadores residenciais individuais em uma rede inteligente de carregamento, integrando:

Balanceamento dinâmico de carga Energia solar fotovoltaica Armazenamento com baterias Comunicação via OCPP Autenticação RFID Gestão inteligente de consumo

O objetivo é tornar a mobilidade elétrica mais sustentável, eficiente e economicamente viável para condomínios.

O Problema

Com o crescimento acelerado dos veículos elétricos no Brasil, condomínios enfrentam diversos desafios:

Sobrecarga da rede elétrica Falta de controle de acesso aos carregadores Dificuldade no rateio do consumo Alto custo energético em horários de pico Falta de integração com energia renovável

O carregador HCA G2, apesar de moderno, foi desenvolvido originalmente para uso individual.

O CollectEV resolve exatamente essa limitação.

Nossa Solução

O CollectEV conecta múltiplos carregadores HCA G2 a um sistema central inteligente (CSMS), permitindo:

Controle individual por usuário Distribuição inteligente de potência Cobrança automatizada por consumo Priorização de energia solar Redução de custos energéticos Monitoramento em tempo real Arquitetura da Solução Painéis Solares ↓ Inversor GoodWe ↓ Baterias GoodWe Lynx ↓ CSMS CollectEV ↓ Rede de Carregadores HCA G2 ↓ Usuários via RFID Tecnologias Utilizadas Tecnologia Função HCA G2 Carregamento EV OCPP 1.6 / 2.0 Comunicação dos carregadores RFID Autenticação de usuários GoodWe ET Inversor solar GoodWe Lynx Armazenamento de energia SEMS Portal Monitoramento energético CSMS Gestão inteligente Funcionamento do Sistema O morador conecta o veículo. O RFID autentica o usuário. O CSMS verifica: energia disponível, geração solar, baterias, sessões ativas. O sistema distribui potência dinamicamente. O consumo é registrado automaticamente. Sustentabilidade

O CollectEV prioriza energia limpa e eficiência energética.

Impactos esperados: Redução de emissões de CO₂ Maior uso de energia solar Redução de desperdícios energéticos Condomínios mais sustentáveis

Um condomínio com 10 EVs pode evitar aproximadamente:

1,26 toneladas de CO₂ por mês Viabilidade

O projeto utiliza tecnologias já disponíveis comercialmente, sem dependência de soluções experimentais.

Modelo de negócio: SaaS para condomínios Cobrança por energia transacionada Integração e instalação da infraestrutura Demonstração
